Methodology
How Strata Works
A continuously running system that assesses how a market price should be handled when it appears in research, dashboards, generated output, or public summaries.
01
Collect Public Market Context

Active Polymarket markets are collected with public-safe context: the selected option, market wording, close timing, thresholds, resolution-source language, category, and public volume.

selected option wording timing thresholds resolution source
02
Route Reference Handling

Strata checks whether the price can survive reuse without losing context that changes how it reads. Disclosure timing, oracle or resolution review, public-health reporting, geopolitical sensitivity, and belief-contamination risks can move a price out of Ready.

disclosure timing oracle review public-health context geopolitical sensitivity belief contamination
03
Downstream Handling

Each market receives a concrete next action for citation and reuse. The status is reference guidance only; it is not outcome prediction, odds validation, market approval, or a trading signal.

Ready Cite with source.
Context Required Attach option, wording, timing, or rule context.
Review Recommended Check rules, timing, or sensitivity first.
