01 - Introduction
What Is Strata
A public market-confidence layer for prediction market prices. Built to help people read the surface more carefully, not to predict outcomes.
"Not every price reflects real consensus."
Strata - core premise
The problem

Prediction market prices are easy to quote and easy to over-read. The same number can sit on top of very different public conditions, and those conditions matter when someone decides how much confidence to place in that price.

Strata exists to make that read more legible. It helps users distinguish between prices that look comparatively steady, prices that are still forming, and prices that deserve more caution.

The key question is not "will this happen?" It is "how should this price be read right now?"

What Strata helps with

Strata adds context to the public market surface through state labels, summary fields, and interpretation cues that help readers understand how much structural confidence to place in a price.

Important distinction. Structural confidence is not the same thing as outcome prediction. A market can look steadier or weaker without answering which side will resolve.
What Strata is and is not
Strata is
A market-confidence layer
A public interpretation surface
A structural reading tool
A way to add context to prices
Strata is not
An outcome prediction engine
A trading or betting signal
A financial advisory product
A guarantee of market truth
Why this matters

When people cite market prices in public, they often treat the number as if it explains itself. That can be too simple. Strata is meant to make the public surface more readable so users can interpret prices with more care.

A price and a strong signal are related, but they are not the same thing. This docs hub exists to explain that difference in plain language.

02 - Dashboard
Reading Market States
Each tracked market is shown through a public state. These labels are meant to help interpretation, not replace judgment.
The three public states
Converged
A comparatively steady public read
The current price appears more settled relative to the rest of the surface. This is the strongest public read shown on the dashboard, but it should still be interpreted as context, not certainty.
A Converged read does not mean the implied outcome will happen.
It means the price signal currently looks more settled than the alternatives.
Markets can move out of this state as public conditions change.
Calibrating
The public read is still forming
The market is still finding its footing. The price may already be useful, but the surface suggests that interpretation should remain more provisional.
This is not automatically a bad state.
It often reflects an active market that is still settling.
Calibrating reads can strengthen or weaken as the surface evolves.
Fragile
Interpret with more caution
The current price should be read more carefully. A Fragile label means the public surface suggests a weaker or less settled read than other markets.
Fragile does not mean the price is false.
It means the public read deserves less structural confidence.
High prices can still appear in Fragile markets.
How to use the labels
State and direction are separate
A market can point strongly in one direction and still deserve caution. The state label describes how the public read should be interpreted, not which side will resolve.
State is a current read
These labels describe how the market appears on the current public surface. They should be read as current context, not permanent tags.
Stronger is not the same as certain
Even the strongest public read remains a market interpretation, not a guarantee. The docs are here to help users read the surface with the right posture.

04 - Interpretation
How to Think About the Surface
This section explains the public interpretation goal behind Strata and how to approach the dashboard with the right posture.
The public interpretation goal

Strata is meant to help users separate two questions that often get blurred together: what direction the market points and how stable that public read looks right now.

Those are related questions, but they are not the same. This docs page exists to make that distinction clearer for anyone reading the public dashboard.

"The public surface is there to help interpret a price more carefully, not to turn a price into a promise."
Strata - interpretation principle
Useful ways to read the surface
Read the state before the number
The state label provides the right posture for interpretation. A price by itself can look more decisive than the surrounding surface really supports.
Use the dashboard as context
The dashboard is most useful when it is read as a context layer. It helps frame the signal rather than reduce the entire market to a single headline price.
Keep confidence and correctness separate
A price can look structurally stronger or weaker without answering whether that price will ultimately resolve in the expected direction.
Prefer interpretation over overreach
The public surface is designed to help users read markets carefully. It should not be treated as a substitute for judgment.
Best use of this page. Read it as a guide to interpreting the dashboard, then return to the live surface with that frame in mind.
