Methodology note

What the labels mean

Strata evaluates reference posture based on public market structure, market wording, resolution sensitivity, and context risk. The status tells a downstream user how a Polymarket price should be handled before it is shown, summarized, cited, or passed into generated output.

  • Ready: standard source attribution is enough. This is not a claim that the odds are correct.
  • Context Required: the price should appear only with the stated context.
  • Review Recommended: the market needs extra scrutiny before downstream use.
  • Not Standalone: do not present the price by itself.
API example

Machine-readable payload

{
  "reference_status": "REVIEW_RECOMMENDED",
  "reference_label": "Review Recommended",
  "reference_action": "Secondary review is recommended before this price is referenced.",
  "reference_reasons": [
    "Resolution or source details need checking before this price is referenced."
  ],
  "disclaimer": "Reference status reflects context risk, not predictive accuracy or market quality."
}
