01
Data Collection

Gamma API pulls 150+ active markets every 30 minutes. CLOB order book depth is fetched for the top 20 markets by volume — replacing volume-only proxies with real bid/ask liquidity. Macro signals (BTC, equities, rates) are captured for correlation analysis.

Polymarket γ-API CLOB depth batch yfinance macro Pydantic validation
02
4-Pillar Analysis Engine
A multi-factor engine that separates real consensus from noise.

Each market is evaluated across four independent dimensions. Category-specific thresholds and hysteresis prevent false state transitions.

Capital · 30pt
CLOB depth → log₁₀(volume) fallback. Real liquidity, not nominal volume.
Efficiency · 25pt
exp(-spread/0.03) × 1/(1+hist_vol×7). Spread quality × hourly volatility.
Independence · 15pt
OLS regression (BTC/SPY/Rates). Penalizes markets driven by macro, not events.
Convergence · 30pt
Normalized entropy decay × time-weighted stability. Genuine consensus lock.
03
State Classification

Nexus Score (0 – 100) maps to one of three states via category-specific thresholds, a 2-hour confirmation window, and hysteresis — preventing oscillation on borderline markets.

Converged Structural consensus locked. Score above threshold, volatility capped, liquidity and independence verified.
Calibrating Price discovery in progress. Market is functional but has not yet reached stable consensus.
Fragile Structural weakness detected. Low liquidity, high spread, or insufficient market depth.
04
Logging & Validation

Every run updates Supabase via race-condition-safe UPSERT (markets_registry, event_log snapshots, convergence_log). Resolved markets feed the track_record table for ongoing Brier Score validation. Dead Man's Switch marks stale markets inactive within 6 hours.
