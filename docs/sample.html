Methodology note

What the labels mean

Strata evaluates reference posture based on public market structure, wording, resolution sensitivity, and context risk. The status tells a downstream user how carefully a Polymarket price should be handled.

  • Ready: standard source attribution is enough.
  • Context Required: the price should travel with the stated context.
  • Review Recommended: secondary review is advised before downstream use.
  • Not Standalone: do not present the price by itself.
API example

Machine-readable payload

{
  "reference_status": "REVIEW_RECOMMENDED",
